# Requirements
* Python 3.6+
* flask
* flask_wtf
* flask_login
* flask_admin
* flask_restful
* wtforms
* sqlalchemy
* sqlalchemy_serializer
# Installation 
Download repo, then go to the repo folder and type it in console
```pip install -r requirements.txt```
# Run app
In project folder open console and run this command ``python main_app.py``
